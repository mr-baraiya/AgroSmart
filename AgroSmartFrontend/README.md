
# AgroSmart

AgroSmart is a smart agriculture management system built with React, Vite, and Tailwind CSS. It helps farmers manage farms, crops, schedules, and weather data efficiently.

## Quick Start

1. **Clone & Install**
   ```bash
   git clone [repository-url]
   cd AgroSmartFrontend
   npm install
   ```
2. **Environment**
   - Copy `.env.example` to `.env` and set:
     - `VITE_API_BASE_URL`
     - `VITE_APP_NAME`
     - `VITE_WEATHER_API_KEY` (optional)
3. **Run**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173)

## Tech Stack

- React 18, Vite, Tailwind CSS
- Context API, Axios, JWT Auth
